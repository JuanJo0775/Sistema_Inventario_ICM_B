**Given (Dado que):**
- Un operario accede al sistema desde una tableta o un teléfono inteligente

**When (Cuando):**
- Navega por cualquier módulo disponible para su rol

**Then (Entonces):**

- Ningún elemento crítico queda oculto o inaccesible en resoluciones móviles
