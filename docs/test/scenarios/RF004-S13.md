## Nombre del test

`tests/ers/test_gherkin_dynamic.py::test_RF004_S13`

## Propósito

## Requisito o caso de negocio asociado

## Inputs (Given / When — extracto ERS)

**When (Cuando):**

- El stock de la ubicación no cambia
